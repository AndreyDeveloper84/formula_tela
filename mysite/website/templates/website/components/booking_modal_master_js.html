{% comment %} JS логика модалки записи к мастеру (#bookingModal). Используется на master_detail.html, home.html и др. Публичные функции: - openBookingModal(serviceId, serviceName, promoOpts?) - openBookingModalWithMaster(serviceId, serviceName, masterName) promoOpts (опционально) для промо-режима: { pinnedOptionId: , // зафиксировать конкретный вариант price: <число>, // фиксированная цена (например, со скидкой) autoPickDate: false // не подставлять первую доступную дату } Когда pinnedOptionId задан — селекты длительности/количества скрываются. Работает с API: - /api/booking/service_options/?service_id=X - /api/booking/get_staff/?service_option_id=X - /api/booking/available_dates/?staff_id=X - /api/booking/available_times/?staff_id=X&date=Y&service_option_id=Z - /api/booking/create/ (POST) {% endcomment %}
